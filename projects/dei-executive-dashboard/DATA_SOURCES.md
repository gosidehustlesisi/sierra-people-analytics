# Data Sources

## Primary Data Sources

| Source | Type | Description | URL |
|--------|------|-------------|-----|
| US Census American Community Survey (ACS) 2022 | Government Dataset | State-level demographic and socioeconomic data | https://data.census.gov/ |
| BLS Employment Data | Government Dataset | Bureau of Labor Statistics employment statistics | https://www.bls.gov/ |

## Data Provenance

- Census ACS 2022 data downloaded from data.census.gov
- BLS employment data fetched from Bureau of Labor Statistics
- Combined for DEI metrics dashboard and workforce diversity analysis

## Refresh Strategy

- Re-download from data.census.gov when new ACS data is released
- Re-fetch BLS data when monthly/annual reports are published
- ACS updates annually; BLS data updates monthly
